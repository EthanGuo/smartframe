expect(self, name=None, interval=None, timeout=None, similarity=0.7, msg=None)

source code 

Allows to search for a expected image in an image file that you provide (e.g. a screenshot taken and saved in a file before) Check whether the expected image was found that satisfy the minimum similarity requirement. If found return self, if not found throw exception.

Parameters:
  • name (string) - The expected image(png) file name.
  • interval (int) - The interval time after one check. Default value is 3 seconds.
  • timeout (int) - The duration time of waitting the screen snapshot. Default value is 9 seconds. See "runner/stability/sysconfig"
  • similarity (float) - The minimum similarity a match should have. If omitted, the default is used
  • msg (string) - Output string if the expected image not found in the target image file.
Returns: testworker.Verify

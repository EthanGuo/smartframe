expect(self, fname, interval, timeout, similarity, msg)

source code 

Allows to search for a expected image in an image file that you provide (e.g. a screenshot taken and saved in a file before) Check whether the expected image was found that satisfy the minimum similarity requirement. If found return self, if not found throw exception.

Parameters:
  • name - The expected image(png) file name.
  • interval - The interval time after one check. Default value is 3 seconds.
  • timeout - The duration time of waitting the screen snapshot. Default value is 9 seconds. See "runner/stability/sysconfig"
  • similarity - The minimum similarity a match should have. If omitted, the default is used
  • msg - Output string if the expected image not found in the target image file.
